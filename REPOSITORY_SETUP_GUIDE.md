# Quick Setup Guide for Data Analytics Portfolio Repositories

This guide will help you create all 6 repositories with the necessary files.

## Step-by-Step Setup Instructions

### Method 1: Using GitHub Web Interface (Easiest)

#### Repository 1: data-analytics-portfolio

1. Go to https://github.com/new
2. Fill in:
   - **Repository name:** `data-analytics-portfolio`
   - **Description:** `Complete portfolio hub for end-to-end data analytics projects`
   - **Visibility:** Public
   - **Initialize:** Add README.md
3. Click "Create repository"
4. Copy the content from `repo-templates/data-analytics-portfolio-README.md` into your README.md

### Method 2: Using Git Command Line

```bash
# Create each repository locally, then push to GitHub

# Repository 1
mkdir data-analytics-portfolio
cd data-analytics-portfolio
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/sivajimenavath/data-analytics-portfolio.git
git push -u origin main

# Repeat for other repositories...
```
